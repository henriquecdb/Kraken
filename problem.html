0001 - Soma Fácil

  • Time Limit: 1.00s
  • Memory Limit: 512 MB

Calcular a soma de dois números inteiros A e B.

Input

O input consiste em dois números inteiros separados por um espaço.

Output

O output deve ser a soma dos dois números.

Example

Input:
2 3

Output:
5
